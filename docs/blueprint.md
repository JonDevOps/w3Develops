# **App Name**: w3Develops

## Core Features:

- User Authentication: Secure user sign-up and sign-in with email/password using Firebase Authentication.
- Profile Management: Enable users to create and edit their profile information, including uploading a profile picture and adding social links.
- Group Sorting: Automatically sort users into learning groups based on their primary skill and learning pace, ensuring group size limits are respected and new groups are created as needed.
- Public Group List: Display a publicly accessible list of all existing groups and their members.
- Form Submission: Handles submission of user sign-up and skill selection forms to correctly sort them.

## Style Guidelines:

- Primary color: Light teal (#A7DBD8), providing a calm and inviting atmosphere suitable for a learning environment.
- Background color: Off-white (#F0F4F3), offering a clean and distraction-free backdrop for content.
- Accent color: Pale green (#90AFC5) to provide visual contrast and highlight important elements.
- Headline font: 'Space Grotesk' sans-serif for headers, giving a modern techy feel. Body Font: 'Inter' sans-serif, used for its readability
- Use clear, consistent icons to represent user actions and navigation elements, improving usability.
- Implement a clean, responsive layout that adapts to different screen sizes and devices.
